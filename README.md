# HR Employee Attrition Analysis

## 📌 Project Overview
This project analyzes employee data to identify key factors influencing employee attrition. The goal is to uncover patterns that organizations can use to improve retention strategies.

---

## 🎯 Objectives
- Identify drivers of employee attrition
- Analyze relationships between job roles, salary, tenure, and overtime
- Perform exploratory data analysis (EDA)
- Provide actionable insights for HR decision-making

---

## 📂 Dataset Information
The dataset includes:

- Age
- Job role
- Department
- Monthly income
- Years at company
- Overtime status
- Job satisfaction
- Attrition status

---

## 🔎 Data Preprocessing
- Handled missing values
- Cleaned inconsistent data
- Converted categorical variables
- Prepared dataset for analysis

---

## 📊 Exploratory Data Analysis (EDA)

### 👩‍💼 Job Role vs Attrition
Certain roles show higher attrition rates.

### 💰 Income Analysis
Lower salary groups show higher attrition probability.

### ⏳ Tenure Impact
Employees with shorter tenure are more likely to leave.

### ⏱ Overtime Effect
Overtime workers have higher attrition rates.

---

## 📈 Key Insights
- Income level strongly influences retention
- Overtime contributes significantly to attrition
- Early-career employees are at higher risk of leaving
- Job satisfaction impacts employee retention
